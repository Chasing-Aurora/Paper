## 难题
### 连接性
- 如果直接 使用 邻接矩阵

## 基于信息传递的逐层传递——🦄

### 针对一个节点的——矩阵去表示
#### 计算公式
- 大量文献认为**列向量**是向量默认的方式，实际上**记住**就行了
	- 既然是向量是列向量，也就是相乘的时候基本在**矩阵的右面**。`m*n`的矩阵乘 `n*1`的向量得到`m*1`的向量。
- ![](../imgs/2026-06-11/file-2026-06-11-205350553.png)
	- A 是**邻接矩阵**就是**按照 key 进行筛选了**，$AH$ 就是 求和邻接节点的Embedding 了！！！
	- D 是**对角矩阵**，$D^{-1}$就是 求平均，除以邻接的数量了！！！


- Half 矩阵就是 开根号

![](../imgs/2026-06-11/file-2026-06-11-212657067.png)

#### Row/Column normalization matrix
- $AH$ 是求**按照key  筛选**后求和
- $D^{-1}A$  是 作用于 $A$，把 **权重平均分配注意到A的每一行**，就是平均 分配个$A$的每一个 连接者
	- 按照 出度平均分配权重 进行归一化
	- 但是 这样只是考虑到了 我的出度，没有考虑到 **对方的出度**！ 所以 右边 再乘以一个去！
- $AD^{-1}$ 是  把 A的每一列 进行 归一化，因为A是对称矩阵，所以就是 上述结果的转置！
- $D^{-1}AD^{-1}$ 是==作用于A，即考虑到了 自己的出度，也考虑到了对方的出度！==总的 分配是 
	- 但是 特征值介于 $-1\sim1$，左乘就会 影响大小，所以再次进行 升级！！！

- **∼（波浪号/相似）**：使用 `\sim`，例如 `$A \sim B$` 显示为 A~B 
- **≃（同构/近似相似）**：使用 `\simeq`，例如 `$A \simeq B$` 显示为 A≃B 
- **≅（全等/几何相似）**：使用 `\cong`，例如 `$A \cong B$` 显示为 A≅B

### Symmetric Normalized Matrix
- $D^{-\frac{1}{2}}AD^{-\frac{1}{2}}$ 保持了最大特征值 是1 ！！！
	- 是 `Symmetric Normalized Matrix`，
	- 也叫做 `Normalized diffusion Matrix`
	- ==其实 **就是邻接矩阵，但是不再是0/1**，而是里面的权重根据连接数进行分配的，就是 开根号再相乘==
		- 而且 **不是 需要学习**的，直接 给定结构 就是固定的
		- ==只有  神经网络的权重，是需要学习的，其他都是不用的==
		- ==这个 是一个**稀疏的矩阵**，所以 可以高效的进行 稀疏矩阵的乘法！==
	- ![](../imgs/2026-06-12/file-2026-06-12-101741663.png)

- 直接 用 Symmetric Normalized Matrix 进行实现了
	- ![](../imgs/2026-06-12/file-2026-06-12-134248001.png)



- 特征值eigenvalue和特征向量eigenvector描述了矩阵变换对向量的影响。
	- ==特征向量在变换过程中保持**方向不变**，而特征值则表示该向量的**缩放因子**==
	- 例如，如果特征值为2，则特征向量的长度会变为原来的两倍；如果特征值为-1，则特征向量的方向会反转，同时长度保持不变

- 半监督的形式  去训练：

	- 编码器 就是 神经网络，就是把 节点变成了 Embedding，==解码器就是 把Embedding 变为数值，比如说是 数量积/内积的计算==



### [inductive learning 与 transductive learning](https://www.zhihu.com/question/68275921/answer/529156908)
- 我们平时所说的learning一般指的是inductive learning
- transductive和inductive的区别在于我们想要预测的样本，是不是我们在训练的时候已经见（用）过的
- 

- GCN、GAT等都是 inductive learning的，可以泛化到 其他节点的，可拓展性 很强
	- pagerank，deepwalk  就是 直推式学习，需要利用  已有的消息 才能进行推理的












## 图神经网路的层数
- 图神经网路的 ==层数 是 计算图的层数==，不是 所有神经网路的层数
	- 因为神经网路 的层数本身 可以达到很多的
	- 就是 k-跳 的网络
- ![](../imgs/2026-06-11/file-2026-06-11-175128212.png)
- GNN  不能无限的深，但是理论是可以的
	- 因为 过深的话，最后 所有节点的计算图 最后都会 很类似的，产生 `over-smoothing` 的问题， 所有节点的 嵌入Embedding  都会 收敛到 同一个值！！！
	- ==所以不能过深，2-3层就是可以的==
- 黑盒子 就是 可以是MLP 的！！！
